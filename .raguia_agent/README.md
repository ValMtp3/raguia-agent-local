Ce dossier contient la configuration locale et les scripts de l’agent.

- **Versionnés dans Git** : `start.sh` / `test.sh` / `stop.sh` (macOS, Linux), `start.bat` / `test.bat` / `stop.bat` (Windows), et les modèles de démarrage automatique générés par l’installateur.
- **Non versionnés** (créés par `install.sh` ou `install.bat`) : `venv/`, `raguia_agent.yaml` (jeton et chemins sensibles).
